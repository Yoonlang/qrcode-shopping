# QRCode Shopping

QR 코드를 스캔해서 원단을 장바구니에 담고 간편하게 주문하세요

[homepage](https://www.jojoywmaeil.com/)

### Skills

#### Frontend

- **Framework**: Next.js (v14)
- **Language**: TypeScript
- **UI Libraries**:
  - React (v18)
  - MUI
  - Emotion
- **State Management**: Recoil
- **Form Handling & Validation**:
  - Formik
  - Yup
- **Internationalization**:
  - i18next
  - react-i18next
- **QR Code & Camera**:
  - jsQR
  - react-webcam
- **File Handling**:
  - jszip
  - xlsx
  - react-pdf
- **Testing**:
  - Jest
  - React Testing Library
  - Cypress

#### Linting

- ESLint
- Husky
- commitlint
- lint-staged

#### Infrastructure & Backend (Separate Privite Repository)

- **Hosting**: Vercel
- **Server**: Node.js
- **Database**: Firebase
